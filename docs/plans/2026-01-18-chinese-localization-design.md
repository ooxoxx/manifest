# 界面中文化设计方案

**日期**: 2026-01-18
**状态**: 已批准

## 概述

将 Manifest 前端界面的所有英文文案直接替换为中文，不引入 i18n 框架。

## 方案选择

**选定方案**: 简单替换（直接修改）

- 直接将英文字符串改为中文
- 不引入国际化框架
- 一次性完成所有文件

## 翻译范围

### 需要翻译

| 类型 | 示例 |
|------|------|
| 页面标题 | Dashboard → 首页 |
| 按钮文字 | Submit → 提交 |
| 表单标签 | Email → 邮箱 |
| 验证错误 | Password is required → 请输入密码 |
| 表格表头 | File Name → 文件名 |
| 提示消息 | Success! → 成功！ |
| 对话框标题 | Add User → 添加用户 |

### 保留英文

- 技术占位符（如 `user@example.com`、`minio.example.com:9000`）
- 专有名词（MinIO、CSV）
- API 字段名
- Access Key / Secret Key
